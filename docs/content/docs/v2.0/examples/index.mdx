# Example Applications

Neogma includes several example applications in the [`/examples`](https://github.com/themetalfleece/neogma/tree/main/examples) directory. Each app demonstrates the same core features using a different model definition style.

## Running an example

All examples require a local Neo4j instance. Start one with Docker:

```bash
docker compose up -d
```

Then run any example:

```bash
cd examples/basic-app-decorators
pnpm install
pnpm start
```

## Available examples

| Example                                                    | Style                                 | Language   |
| ---------------------------------------------------------- | ------------------------------------- | ---------- |
| [TC39 Decorators](/docs/v2.0/examples/decorators)          | `@Node`, `@Property`, `@Relationship` | TypeScript |
| [Legacy Decorators](/docs/v2.0/examples/legacy-decorators) | `experimentalDecorators`              | TypeScript |
| [JavaScript](/docs/v2.0/examples/javascript)               | `ModelFactory`                        | JavaScript |
| [NestJS](/docs/v2.0/examples/nestjs)                       | `@neogma/nest` module                 | TypeScript |
